📚 Library Management System in C

A simple Library Management System built using the C programming language. This project demonstrates core concepts of file handling, structures, and menu-driven programming.

🚀 Features
➕ Add new books

📖 Display all books

🔍 Search book by ID

🔢 Count total number of books

📂 Persistent storage using file handling (inventory.txt)

🛠️ Technologies Used

Language: C
Concepts:

Structures

File Handling (fopen, fscanf, fprintf)

Functions
Menu-driven program
